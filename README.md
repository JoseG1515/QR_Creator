# QR_Creator
este es un creador de codigo QR, el mismo genera QR a partir de texto
Stack Tecnologico: React.Js + vite para el desarrollo del Frontend, Python- Fastapi para el desarrollo del Backend donde se genera el codigo QR. 
PASOS PARA CORRER ESTE PROYECTO EN LOCAL  
1- descargar el proyecto en tu computadora 
2- abrirlo en el visual studio code 
3- abrir una terminar y dirigirte la carpta "Frontend" utilizando el comando cd
4-luego ejecutar en el terminal el comando "npm install" para instalar las dependencias necesarias para que el frontend funcione
5-ejecutar el comando "npm run dev" este comando ejecuta nuestro frontend de manera local en la terminal podremos ver que se imprime algo como " VITE v7.3.1  ready in 467 ms

  ➜  Local:   http://localhost:5173/
  ➜  Network: use --host to expose
  ➜  press h + enter to show help"
  la podemos ver nuestro frontend en funcionamiento copiando la url local en el navegador
6-abrir otra terminal y dirigirse a la carpeta Backend utilizando el comando cd 
7-ejecutar el comando "pip install -r requirements.txt" para instalar las dependencias de python
8-una vez se instalan las dependencias, podemos ejecutar "uvicorn Server:app --reload --port 8000" este comando ejecuta nuestro Backend en el puerto 8000 

CONCEPTOS QUE REFORCÉ CON ESTE PROYECTO
1- Consumo de APIs
2- Separación frontend/backend 
3- Generación y manejo de archivos desde el servidor
4- Buenas prácticas de estructura de proyecto
